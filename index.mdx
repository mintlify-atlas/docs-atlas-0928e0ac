Ask Nova is the AI support backend powering SoftSUITE Novo's intelligent chat assistant. It combines a FastAPI REST interface, LanceDB hybrid vector search, HuggingFace embeddings, and the Groq Llama 3.3 70B language model to deliver accurate, knowledge-grounded answers from your own document library.

## How it works

Ask Nova uses Retrieval-Augmented Generation (RAG) to answer questions. When a user sends a message, the API retrieves the most relevant document chunks from a LanceDB vector store, then passes them as context to the Groq LLM. This ensures answers are grounded in your official documentation rather than general training data.

<Steps>
  <Step title="Configure your environment">
    Set `HF_TOKEN`, `HUGGINGFACE_API_KEY`, and `GROQ_API_KEY` in a `.env` file. See [Environment Variables](/configuration/environment) for the full list.
  </Step>
  <Step title="Add your documents">
    Place PDF, TXT, or Markdown files in a `docs/` folder next to `app.py`. The API ingests them automatically on startup.
  </Step>
  <Step title="Start the server">
    Run `uvicorn app:app --host 0.0.0.0 --port 8000`. The knowledge base is indexed and the API is ready.
  </Step>
  <Step title="Send a chat request">
    POST to `/chat` with a `message` field. Receive a grounded answer and a `session_id` to continue the conversation.
  </Step>
</Steps>
